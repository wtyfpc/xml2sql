# 使用说明

xmltosql.jar工具放到metrics目录下，执行命令：

`java -Dfile.encoding=UTF-8 -jar xmltosql-1.0-SNAPSHOT.jar` 

会在metrics目录下生成data目录，里面存放日志以及sql文件。

`-Dfile.encoding=UTF-8` 参数用于解决编码问题。

```
|_metrics
	|_xmltosql.jar
	|_antiddos
	|_appgroup
	|_bmc
	|_linuxvm
	....

执行 java -Dfile.encoding=UTF-8 -jar xmltosql-1.0-SNAPSHOT.jar，会为上面的各目录中的*metric_template.xml模板文件生成sql。执行后的目录层级如下：

|_metrics
	|_xmltosql.jar
	|_antiddos
	|_appgroup
	|_bmc
	|_linuxvm
	|_data
		|_log
		|_sql
	....
```
